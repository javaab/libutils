libutils
===============================
An utility library


The documentation can be found at: https://javaab.github.io/libutils-docs/


Installation
------------
Install via setuptools:

.. code-block:: console
    
    python setup.py install

How to use
-----------------------
Add ``django_accounts`` to your configuration file (normally settings.py): 

.. code-block:: python

    import libutils
