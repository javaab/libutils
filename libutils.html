libutils.log_util module¶

Logger Utility

Set ups logger to ‘newtonsystems’ style

Usage:

from libutils import log_util

log_util.get_logger(__name__)

libutils.log_util.get_logger(logger_name)¶

Provides a logger that supports our prefixing code. If your code needs to use [push|pop]_prefix or calls out to code that does then you should use this to get your logger:

logger = log_util.get_logger(__name__)

As opposed to:

logger = logging.getLogger(__name__)
